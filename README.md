# Ryze Website Redesign

A modern, interactive redesign of the Ryze AI advertising platform website, built with Next.js 14, TypeScript, and Tailwind CSS. Features unique emerald/amber color palette, interactive components, and motion backgrounds.

## Live Demo

[Deploy to Vercel](https://vercel.com) - Simply connect your GitHub repository and deploy.

## Project Overview

This project is a complete redesign of the Ryze website (get-ryze.ai) with improved visual appeal, user experience, and interactive features. The design features a unique emerald/amber color palette, motion backgrounds, and interactive components like creative analyzer, account audits, and AI chat interface.

##  Design Decisions

### Visual Design
- **Color Palette**: Unique emerald (#10b981) and amber (#f59e0b) gradients create a fresh, energetic aesthetic
- **Typography**: Inter font family for clean, readable text with clear hierarchy
- **Motion**: Advanced particle effects and animated backgrounds
- **Interactivity**: Live components like creative analyzer, budget suggestions, and AI chat

### User Experience
- **Interactive Features**: Creative analyzer with scoring, account audit demos, budget reallocation suggestions
- **AI Chat Interface**: ChatGPT-like interface for asking questions about campaigns
- **Platform Connections**: Animated connection buttons for Google Ads, Meta, Amazon, etc.
- **Signup Flow**: "Get Started" redirects to Google OAuth signup page

##  Technology Stack

- **Next.js 14**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **Framer Motion**: Advanced animations and transitions
- **Lucide React**: Modern icon library

##  Installation

```bash
npm install

npm run dev


npm start
```

##  Interactive Features

- **Creative Analyzer**: Upload and analyze ad creatives with CTA, Visual, and Copy scoring
- **Account Audit**: Interactive demo showing campaign optimization suggestions
- **Budget Reallocation**: Live suggestions for budget optimization
- **AI Chat**: ChatGPT-like interface for campaign questions
- **Platform Connections**: Animated connection buttons for all major ad platforms
- **Motion Backgrounds**: Particle effects and animated gradients
